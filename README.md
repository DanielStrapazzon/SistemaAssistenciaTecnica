**Sistema de Assistência Técnica**

- Este sistema deve permitir o cadastro de tipos de serviço, como manutenção, reparo e diagnóstico, onde cada tipo possui
um valor por hora ou valor fixo.

- A entidade principal será o registro de chamados, contendo informações como cliente, descrição do problema, data e
hora de abertura e de conclusão do atendimento. Cada chamado deve estar vinculado a um tipo de serviço.

- A regra de negócio consiste em calcular o tempo total de atendimento com base nas datas registradas e calcular o
valor final conforme o tipo de serviço escolhido.
